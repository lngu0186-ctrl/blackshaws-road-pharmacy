## Key Learnings
- ServiceDetail component can dynamically handle: hub overview, real services (from services.ts), mapped slugs (via serviceSlugMapping), and stub fallback — all in one component
- Meta description injection via useEffect in page components works reliably for Lovable hosting
- Supabase Edge Functions are the cleanest path for form submission without adding new deps
- The services.ts dataset was comprehensive but completely disconnected — routing was the main architectural fix needed
